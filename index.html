Mengenal Data Flow Diagram (DFD)

A. Apa itu Data Flow Diagram?

Data Flow Diagram (DFD) adalah alat visual yang digunakan untuk menggambarkan aliran data dalam sistem informasi. DFD membantu dalam memahami bagaimana data bergerak dalam sebuah sistem, dari input hingga output.

C. Jenis-Jenis DFD

DFD memiliki beberapa level, antara lain:

  • DFD Level 0 (Context Diagram)
  • DFD Level 1
  • DFD Level 2
  • Dan seterusnya

D. Cara membuat DFD

Langkah-langkah membuat DFD meliputi:

  1. Identifikasi entitas dan proses utama.
  2. Tentukan aliran data antara entitas dan proses.
  3. Gambar diagram menggunakan simbol-simbol standar DFD.
  4. Verifikasi dan validasi diagram.
